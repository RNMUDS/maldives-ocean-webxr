# Maldives Ocean WebXR

Three.js 製のモルディブ水上ヴィラリゾート WebXR コンテンツ。
ターコイズのラグーンに茅葺き屋根の水上ヴィラが並び、インフィニティプールから水がこぼれ、
熱帯魚・マンタ・ウミガメが回遊し、食物連鎖まで起きる海を、デスクトップでも VR でも歩き回れます。

## 主な機能

- **海と空**: Water シェーダー（反射率・散乱をカスタムパッチ）＋ Sky 大気散乱、頂点シェーダーによる波のうねり、半透明の水越しに見える砂の海底
- **水上ヴィラ 6棟**: 縦張りホワイトウッドの壁・茅葺き屋根（プロシージャル生成 PBR テクスチャ）、ガラス引き戸、インフィニティプール（越流エッジ＋落水＋波紋）
- **海洋生物**: カラフルな熱帯魚600匹（InstancedMesh）、魚群、跳ねる魚、マンタ、ウミガメ、中型魚→大型魚の捕食 AI（食物連鎖）
- **巨大スクリーン**: 沖合の大型ビジョンに `getDisplayMedia` で自分の画面をリアルタイム投影
- **WebXR**: VR ヘッドセット対応（コントローラ移動）。HTTPS 環境が必要

## 操作

| 操作 | 内容 |
|---|---|
| クリック | 一人称視点開始（マウスで見回し / ESC で解除） |
| WASD・矢印キー | 移動 |
| Shift | 高速移動 |
| Space | 押している間上昇、離すと下降 |
| 右上ボタン | 巨大スクリーンへの画面共有 |

## 実行

```bash
python3 -m http.server 8765
# → http://localhost:8765/
```

VR（Meta Quest 等）で試す場合は HTTPS でホストしてください。

## テクスチャの再生成

藁葺き・漆喰・ホワイトウッド・コンクリートの PBR テクスチャは Python スクリプトで生成しています。

```bash
python3 tools/gen_textures.py  # 要 numpy / Pillow
```

## 構成

```
index.html        エントリ
js/main.js        初期化・操作系・画面共有ボタン
js/ocean.js       海・空・太陽・環境プリセット
js/resort.js      ヴィラ・桟橋・プール・落水
js/marinelife.js  海洋生物と食物連鎖 AI
js/clouds.js      積雲
js/screen.js      沖合の巨大スクリーン（画面共有）
js/textures.js    テクスチャローダー
tools/gen_textures.py  PBR テクスチャ生成
textures/         生成済みテクスチャ
```
